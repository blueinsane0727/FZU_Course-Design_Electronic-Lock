# 基于51单片机的电子密码锁

纪念完成的第一个比较完整的项目，本质是一个没什么技术含量的课设，代码也只做到了能跑，有很多需要完善的地方。

## 概述

本项目是一个基于 stc89c52rc 的电子密码锁，它通过4x4矩阵键盘输入密码，在数码管上显示交互信息，并使用电机模拟控制锁具的开关。系统具备密码验证、修改和错误报警等基本功能，是FZU的微处理器原理及应用课程设计的作品。

## 核心功能

- **密码验证**： 输入3位密码，验证正确则开锁。
    
- **密码修改**： 在验证旧密码成功后，允许用户设置新密码，并掉电保存
    
- **交互显示**： 使用数码管实时显示输入状态和操作提示
    
- **错误报警**： 连续错误输入超过5次，系统报警播放春日影（不完整版）。
